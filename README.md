# HRMS React Dashboard

A full-stack Human Resource Management System dashboard built with React.js.

## Features
- Employee directory with search and department filter
- Leave request management with approve/reject
- Real-time attendance tracking
- Role-based status indicators

## Tech Stack
- Frontend: React.js, JavaScript
- Backend: Node.js, PostgreSQL
- Auth: Microsoft Azure AD SSO
